# PayPay API セットアップ手順

## 1. 加盟店審査（1〜2週間）
1. https://developer.paypay.ne.jp/ にアクセス
2. 「加盟店申請」→ 事業内容・URL（kairox.jp）を入力
3. 開業届 or 法人登記が必要（個人事業主OK）
4. 審査通過後 → API_KEY / API_SECRET / MERCHANT_ID を取得

## 2. 環境変数
```
PAYPAY_API_KEY=your_api_key
PAYPAY_API_SECRET=your_api_secret
PAYPAY_MERCHANT_ID=your_merchant_id
PAYPAY_ENV=production  # or sandbox
```

## 3. SDK インストール
```bash
npm install paypayopa-sdk-node
```

## 4. 支払いフロー（QRコード）
```typescript
import PayPay from 'paypayopa-sdk-node';

const client = PayPay({
  clientId: process.env.PAYPAY_API_KEY!,
  clientSecret: process.env.PAYPAY_API_SECRET!,
  merchantId: process.env.PAYPAY_MERCHANT_ID!,
  productionMode: process.env.PAYPAY_ENV === 'production',
});

// Dynamic QR生成
const qr = await client.QRCodeCreate({
  merchantPaymentId: bookingId,
  amount: { amount: total, currency: 'JPY' },
  codeType: 'ORDER_QR',
  orderDescription: `KAIROX 荷物配送 ${bookingId}`,
  isAuthorization: false,
  redirectUrl: `https://kairox.jp/narita?paid=${bookingId}`,
  expiredAt: Math.floor(Date.now() / 1000) + 900, // 15分有効
});
```

## 5. 手数料
- 決済手数料: 1.98%
- 月額: 無料
- 最低取引額: なし

- 開業届提出後（6/8以降）に申請
