### 3. `/driver` ルート新設（ドライバー専用画面）

| フライトボード | 成田空港到着便リアルタイム表示 |

**アクセスURL:** `https://frontend-psi-seven-15.vercel.app/driver`

### 4. 予約確認メール送信機能
`commit 3713f63`

**変更ファイル:** `app/narita/page.tsx`
**新規ファイル:** `app/api/send-confirmation/route.ts`

#### フロントエンド変更

Confirmステップにメールアドレス入力欄を追加（任意項目・4言語対応）:

| 言語 | ラベル | プレースホルダー |
|------|--------|----------------|
| EN | Email (optional) | your@email.com |
| JA | メール（任意） | your@email.com |
| ZH | 邮箱（选填） | your@email.com |
| KO | 이메일 (선택) | your@email.com |

入力欄の下に「予約確認・追跡URLをお送りします」の説明テキスト表示。

#### バックエンド変更

`/api/booking` POST に `email` フィールドを含めるよう変更（以前は空文字固定）。

#### 確認メールAPI

`POST /api/send-confirmation`

- Resend HTTP API を直接呼び出し（SDK不要）
- `RESEND_API_KEY` 未設定でも予約フローをブロックしない（graceful degradation）
- 4言語対応のHTMLメール（予約ID・届け先・合計・追跡URLを含む）
- 追跡URL: `/narita/status/{booking_id}`

**本番有効化に必要な Vercel 環境変数:**
```
RESEND_API_KEY=re_xxxxxxxxxx
RESEND_FROM=KAIROX <noreply@kairox.app>
```
Resend 無料枠: 3,000通/月
