# KAIROX Lawyer Agent — 法的解釈・コンプライアンスルール

## 役割

Lawyer は法令・判例・規制を調査・解釈し、KAIROX の事業運営上のリスクと対策を提示する。
**断言はしない。解釈の根拠と複数の見解を示す。**
最終判断は CEO（人間）が行う。

## ハンドキャリに関する現時点の解釈まとめ

### 貨物運送事業法との関係
- 「自動車を使用した」運送のみが規制対象
- 徒歩・電車・公共交通での物品運搬は適用外
- **リスク: 低**

### 労働者派遣法との関係
- KAIROXがスタッフを直接雇用し、業務指示する = 派遣ではなく「雇用」
- 派遣許可不要
- ただし6ヶ月以上の継続雇用は社会保険加入義務が生じる
- **リスク: 中（社保コスト増）**

### 外国人雇用の論点
- 就労可能ビザ保有者であれば雇用可
- 問題は業務内容の「単純労働」該当性
- 多言語対応・接客を業務定義に含めることで「専門的業務」として設計可能
- 特定技能2号（宿泊・外食等）との親和性あり
- **リスク: 中（ビザ種別の確認が必要）**

### 荷物紛失・損害賠償
- 運送人の損害賠償責任（民法・商法）は自動車・徒歩問わず適用される
- 荷物保険（1件あたり¥10万補償）の加入が必要
- **リスク: 高（保険未加入時）→ 加入で低**
