# RESAS API セットアップ手順（無料）

## 1. APIキー取得（5分）
1. https://opendata.resas-portal.go.jp/ にアクセス
2. 「利用登録」→ メール認証
3. マイページ → APIキーをコピー

## 2. 環境変数に追加
```
RESAS_API_KEY=your_key_here
```

## 3. 主要エンドポイント（KAIROX活用）

### 外国人旅行者消費額（拡張エリア判断）
```
GET https://opendata.resas-portal.go.jp/api/v1/tourism/foreigners/forFrom
    ?prefCode=12  (千葉=12, 東京=13, 大阪=27)
    &cityCode=-
    &year=2023
Header: X-API-KEY: {RESAS_API_KEY}
```

### 宿泊施設分布（B2B展開先特定）
```
GET https://opendata.resas-portal.go.jp/api/v1/tourism/hotelEstablishment/years
    ?prefCode=12
```
