## 価格設計の出力フォーマット

```
### 価格案
- 基本料: ¥X,XXX（荷物1個・XX km以内）
- 追加荷物: ¥XXX/個
- 遠距離追加: ¥XXX/km

### 根拠
- 競合比較: [競合A ¥X,XXX / 競合B ¥X,XXX]
- 原価試算: [ドライバー時給・燃料・保険]
- 想定利益率: X%

### リスク
- [価格が高すぎると...] [安すぎると...]
```

## 給付金受給中の注意事項

- 売上が発生した場合、**その都度ハローワークに届出必須**
- 就職内定時は就業開始日を即報告
- 副業所得が月20万円を超えると確定申告義務
