## v0.5.0 — 2026-03-20

### ドライバーGPS追跡 + 日本国内旅行者対応

**新機能: ドライバーGPSリアルタイム追跡**
- `DriverView` — ドライバー専用画面（PIN認証: 4桁）
  - 追跡番号を入力して配送を管理
  - GPS追跡トグル（`navigator.geolocation.watchPosition` + 30秒ごと自動送信）
  - ステータス更新ボタン：向かっています → 近くにいます → 集荷中 → 配達完了
  - ステータス変更時に即時GPS位置送信
  - 複数配送を同時管理可能
- ヘッダーナビゲーションに🚐ボタン追加（ドライバー専用、目立ちにくい配置）
- `TrackingView` — ドライバー位置情報カードを追加
  - バックエンドポーリング（10秒ごと）
  - ドライバーとの距離表示（haversine計算）
  - ETA推定（平均30km/hベース）
  - 距離バー（集荷地点→配送先のプログレスバー）
  - LIVEパルスアニメーション
  - ドライバーステータスバッジ（向かっています / 近くにいます / 到着）
- バックエンド `PUT /bookings/{id}/driver-location` エンドポイント追加
  - BookingRecordに `driver_lat`, `driver_lng`, `driver_status`, `driver_updated_at` フィールド追加
  - ドライバーステータスに応じてBookingのstatusも自動更新
  - `GET /bookings/{id}` レスポンスにdriver情報を含める
- Next.js `/api/booking` にPUTプロキシハンドラ追加

**新機能: 日本国内旅行者対応**
- ヘッダーに「国内旅行 OK」バッジを追加（バイオレット配色）
- 国内旅行者向けの差別化ポイント：
  - JPYC（1 JPYC=¥1）で為替リスクなし、両替不要
  - 日本語UI完全対応
  - 千歳・札幌・小樽の国内観光スポット対応
  - 国内法人（ラピダス等）はJPYC + 法人プランで経費処理しやすい

## v0.4.0 — 2026-03-20

### JPYC決済対応 + 決済UI刷新

**新機能**
- **JPYC（日本円コイン）on Polygon** 決済に対応
  - 1 JPYC = ¥1 固定レート。為替リスクなし、0% 手数料
  - 法的分類：前払式支払手段（資金決済法）。暗号資産非該当
  - 支払い金額がそのままJPYC数量で表示（換算不要）
  - Polygonネットワーク対応QRコード・ウォレットアドレス表示
- 支払い方法を3択に拡張：クレジットカード / JPYC / USDC
  - JPYCタブはバイオレット配色で差別化
- 料金サマリーにJPYC・USDC両方の等価額を同時表示
- AIチャットのシステムプロンプト更新（JPYC・最新料金・スポット情報を反映）

**事業計画書更新（v1.2）**
- 第11章「JPYC × USDC 決済戦略 — メディア・PR角度」を追加
  - PRプレイブック（JPYC Inc.提携・メディアリリース先・ストーリーライン）
  - 「競合が真似できない理由」分析

## v0.2.0 — 2026-03-20

### AI相乗りマッチングエンジン（バックエンド）

**新機能**
- `POST /bookings` エンドポイント — 予約作成 + 即時AIマッチング実行
- `GET /bookings/{id}` エンドポイント — 予約情報・マッチ状態取得
- `backend/matching.py` — 相乗りマッチングエンジン
  - 同スロット × 同ゾーン × 相乗りON × 未マッチの予約を自動グルーピング（最大3件）
  - グループ成立時に Claude haiku-4-5 へ最適集荷順とETA（分）を問い合わせ
  - 失敗時はデフォルト順にフォールバック（サービス継続性を保証）
- `BookingRecord` テーブル（SQLite/PostgreSQL）に予約データ永続化
- Next.js `/api/booking` プロキシルート（CORS・認証ハンドリング）
- フロント: 支払い確定時にAPIを呼び出し、実際のトラッキングIDを取得
- 確定画面にマッチ結果表示（マッチ人数・推定所要時間・Claudeの最適化理由）
- 送信中スピナー表示 / APIエラー時はクライアント側トラッキングIDにフォールバック

## v0.1.0 — 2026-03-20

### MVP フロントエンド + 料金・UX機能群

**新機能: 料金体系**
- Hokkaido MVP 料金テーブル（Solo / Pair / Family × chitose / sapporo / otaru）
- 急行保証料 +¥1,000（スロット選択時のみ加算）
- 相乗り割引 −15%（小計全体から計算）
- GPS空港割引 −¥500（新千歳空港15km圏内を自動検出）
- 料金内訳をStepPaymentで全項目表示（基本料・急行料・追加荷物・各割引・合計）
- 節約額をグリーンで強調表示

**新機能: 急行便スロット**
- FLIGHT_SLOTS — 国際線フライト時刻に対応した4便
  - 便①〜④（仁川・香港・台北便に連動）
- フライト番号入力 → 推奨スロット自動提案（`suggestSlot`）
- 推奨スロットに「おすすめ」バッジ表示

**新機能: 相乗りマッチングUI**
- 相乗りトグルボタン（緑色アクティブ）
- マッチング中アニメーション（パルス + AIバッジ）
- 「2名が同方向に予約中」表示

**新機能: ホテル検索**
- ゾーン別ホテルリスト（千歳11件・札幌12件・小樽7件）
- 全ゾーン横断検索対応
- フリーテキスト入力フォールバック

**新機能: キャンセルポリシー（確定画面）**
- 予約から10分以内: 全額返金

**新機能: 法人向けページ（BusinessView）**

**新機能: AIチャットサポート（ChatWidget）**

**エリア**
