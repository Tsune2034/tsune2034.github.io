## 会議サマリー

### market（市場）

**訪日4,268万人市場の現実：**
- 荷物が「電車の通路を塞ぐ」問題が社会課題化
- 既存サービス（ヤマト・JAL ABC）はすべてカウンター型・前日締め切り
- Bounce×Yamatoが参入したことは「市場が本物」の証明
- **GPSオンデマンド（ドライバーが来る）型は依然として空白**

**訪日市場の変化（2026年）：**
- 韓国人旅行者が最大勢力（+22%・月117万人ペース）→ KO対応済みのKAIROXが有利
- 中国人旅行者は−61%の急落 → ZH依存は危険
- 東南アジア・中東・ヨーロッパからの増加が補完

### Bounce×Yamatoとの差別化（重要）

| Bounce×Yamato | KAIROX |
|--------------|--------|
| 翌日〜翌々日配送 | **当日・即時配送** |
| ロッカー・カウンターへ行く | **ドライバーが来る** |
| 24時間対応不可 | **深夜便・遅延便対応** |
| 多言語UI なし | **4言語完全対応（EN/JA/ZH/KO）** |
| AIチャット なし | **Claude AIサポート付き** |

**結論：彼らが参入したことは市場の本物性の証明。彼らが埋められない穴を集中して攻める。**
