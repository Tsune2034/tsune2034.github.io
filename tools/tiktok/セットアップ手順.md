# TikTok API セットアップ手順（@channel.yari 収益化済み）

## 1. Developer App作成（申請制）
1. https://developers.tiktok.com/ にアクセス
2. 「Create App」→ 用途: 「Content Automation for Creator」
3. 必要なScope:
   - `video.upload` — 動画投稿
   - `video.publish` — 公開
   - `user.info.basic` — プロフィール情報
4. 審査期間: 1〜4週間（2025年〜厳格化）

## 2. 必要なもの
- kairox.jp のプライバシーポリシーURL
- @channel.yari のCreatorアカウント（収益化済みなのでSandbox制限が緩い）
- 投稿目的の明示（日本の旅行情報コンテンツ）

## 3. 環境変数
```
TIKTOK_CLIENT_KEY=your_client_key
TIKTOK_CLIENT_SECRET=your_client_secret
TIKTOK_ACCESS_TOKEN=user_access_token  # OAuth後に取得
```

## 4. n8n自動投稿ノード（審査通過後）
```json
{
  "type": "n8n-nodes-base.httpRequest",
  "parameters": {
    "url": "https://open.tiktokapis.com/v2/post/publish/video/init/",
    "method": "POST",
    "headers": { "Authorization": "Bearer {{ $env.TIKTOK_ACCESS_TOKEN }}" }
  }
}
```
