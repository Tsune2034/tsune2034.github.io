## PIN 設定場所

Vercel Dashboard → Settings → Environment Variables

| 変数名 | 用途 |
|--------|------|
| `DRIVER_PIN` | ドライバーポータル（/driver） |
| `ADMIN_PIN` | 管理画面（/admin） |
| `PLAYER_PIN` | プレイヤー画面（/player） |

---

## テスト手順

1. **管理画面でテスト予約を作成**
   - `/admin` にアクセス → ADMIN_PIN 入力
   - 「テスト予約作成」フォームで予約ID を発行

2. **ドライバーポータルにログイン**
   - `/driver` にアクセス → DRIVER_PIN 入力

3. **配達ステータスを進める**
   - 向かっています → 近くにいます → 荷物を受け取り中 → 配達完了

4. **証跡写真を撮影**
   - 受取時・配達完了時にそれぞれ写真アップロード

5. **追跡URLを確認**
   - `/narita/status/[予約ID]` で顧客側の表示を確認

6. **気づいた問題をメモ**
   - 帰宅後に `agents/tasks.json` にタスク追加
