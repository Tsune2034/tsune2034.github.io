# e-Stat API セットアップ手順（無料）

## 1. appID取得（3分）
1. https://www.e-stat.go.jp/api/api-info/api-guide にアクセス
2. 「ユーザ登録」→ メール認証
3. 「APIキー発行」→ `appId` を取得

## 2. 環境変数に追加
```
ESTAT_APP_ID=your_app_id_here
```
Railway（バックエンド）と n8n（環境変数）両方に設定。

## 3. n8nワークフローを有効化
`tools/n8n/workflows/kairox_estat_monthly.json` をインポート。
毎月1日9時JST に訪日外客数をTelegramに自動送信。

## 4. 使えるデータセット
| statsDataId | 内容 |
|---|---|
| `0003448896` | 訪日外客数（法務省） |
| `0003148163` | 宿泊旅行統計（ホテル稼働率）|
| `0003412536` | 外国人延べ宿泊者数（都道府県別）|
