# Aviation Edge — 成田空港フライト連動

## 概要
フライト遅延を検知 → 顧客への自動通知・配達スロット調整

## 料金
- Starter: $149/月（1,000リクエスト/日）
- Business: $299/月（10,000リクエスト/日）
- **導入タイミング: 月次売上が¥10万超えた時点で検討**

## 環境変数（取得後）
```
AVIATION_EDGE_API_KEY=your_key
```

## 使用するエンドポイント
```
# 成田空港 到着便リアルタイム
GET https://aviation-edge.com/v2/public/flights
    ?key={API_KEY}
    &airportIata=NRT
    &status=landed
    &direction=arrival

# フライト詳細（遅延時刻）
GET https://aviation-edge.com/v2/public/flightsFuture
    ?key={API_KEY}
    &iataNum=NH101  (便名)
    &type=departure
```

## n8nワークフロー（導入後）
- フライト到着予定1時間前 → 顧客にTelegram/WhatsApp通知
- 遅延検知 → 「XX便が遅延しています。配達スロットを自動調整します」
- 早着検知 → ドライバーに早めの出発アラート
