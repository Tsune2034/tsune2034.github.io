# Stripe Japan 決済拡張 — セットアップ手順

## 1. Konbini払い（コンビニ支払い）

### Dashboard設定（5分）
1. https://dashboard.stripe.com/settings/payment_methods にアクセス
2. 「Konbini」を検索 → 有効化
3. 対応コンビニ: ローソン・ファミマ・ミニストップ・セイコーマート

### コード対応（支払い期限 = 3日間）
バックエンド `backend/main.py` の `create-intent` エンドポイントに追加:

```python
# konbini 支払い
if pay_method == "konbini":
    intent = stripe.PaymentIntent.create(
        amount=amount,
        currency="jpy",
        payment_method_types=["konbini"],
        payment_method_options={
            "konbini": {
                "expires_after_days": 3,
                "product_description": f"KAIROX 荷物配送 {booking_id}",
            }
        },
        metadata={"booking_id": booking_id},
    )
```

### 環境変数（必要なし — 既存STRIPE_SECRET_KEYで動作）

## 2. 銀行振込（Furikomi）— B2Bホテル向け

### Dashboard設定
1. https://dashboard.stripe.com/settings/payment_methods
2. 「Bank transfer（JPY）」を有効化
3. Billing → Invoices → 自動照合を有効化

### 用途
- ホテルパートナーへの月次まとめ請求
- 手動振込→自動照合で経理工数ゼロ

---

## 3. 必要な環境変数

| 変数名 | 説明 |
|---|---|
| `STRIPE_SECRET_KEY` | 本番キー（sk_live_...）|
| `STRIPE_PUBLISHABLE_KEY` | 公開キー（pk_live_...）|
| `STRIPE_WEBHOOK_SECRET` | Webhook署名検証 |
