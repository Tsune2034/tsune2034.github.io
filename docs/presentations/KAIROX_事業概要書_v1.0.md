## 2. 課題とソリューション

### 課題：訪日外国人が直面する「荷物問題」

訪日旅行者の最大のストレスは **空港到着直後の重い荷物** です。

```
成田到着 → 重いスーツケースを引きながら電車 → ホテルチェックイン前は荷物を持ち歩く
  ↓
観光・ショッピング・グルメ全てが制限される
```

**データで見る課題の大きさ：**
- 訪日外国人数：2024年 **3,687万人**（過去最高・JNTOデータ）
- 成田空港利用者：年間 **約1,700万人**（国際線）
- コインロッカー費用：¥500〜¥1,000/日 × 滞在日数
- 手荷物配送既存サービス（ヤマト等）：空港カウンター締め切り制・翌日配送

### ソリューション：KAIROX

```
空港到着 → スマホでQRスキャン → 荷物を渡すだけ（3分）
  ↓
当日中にホテルへ配送（リアルタイムGPS追跡）
  ↓
チェックイン前に荷物が到着 → 手ぶらで全力観光
```

**KAIROXが選ばれる3つの理由：**

| 比較軸 | コインロッカー | ヤマト宅急便 | **KAIROX** |
|---|---|---|---|
| 当日配送 | ❌（預けるだけ） | ❌（翌日〜） | ✅ **当日配送** |
| ホテル直接配達 | ❌ | △（翌日） | ✅ **チェックイン前OK** |
| スマホ完結 | ❌ | ❌（カウンター） | ✅ **QRのみ** |
| リアルタイム追跡 | ❌ | △ | ✅ **GPS追跡** |
| 多言語対応 | ❌ | ❌ | ✅ **4言語** |
| 価格 | ¥500〜1,000/日 | ¥1,400〜2,500/個 | **¥2,500〜3,000/個** |

## 3. サービス概要

### ユーザーフロー

```
① 成田空港到着
   └ QRコードをスキャン（ターミナル1/2/3対応）

② 預け場所を選択（4言語対応アプリ）
   └ スタッフ不要・スマホのみで完結

③ 荷物を渡す（3分）
   └ ドライバーが空港で直接受け取り

④ リアルタイムGPS追跡
   └ 荷物の位置がスマホで確認できる

⑤ ホテルチェックイン前に配送完了
   └ フロントまたは部屋に直接届ける
```

### 料金体系

| プラン | 料金 | 内容 |
|---|---|---|
| スタンダード | ¥2,500/個 | 成田→千葉・成田市内ホテル |
| シティ | ¥3,000〜7,000/個 | 成田→東京各エリア（距離別） |
| エクスプレス | +¥1,000 | 優先配送（3時間以内） |
| グループ割 | 10%OFF | 3個以上 |

### 配送エリア（現在）

- 成田市内ホテル（¥2,500〜）
- 千葉市・幕張・海浜幕張（¥4,800〜5,200）
- 東京都内全域（¥6,500〜8,500）
- 埼玉・さいたま（¥8,500）
- 横浜（¥9,500）
- 羽田空港T1/T2/T3（¥9,500）

## 5. 配送エリア戦略

### Phase 1: 千葉エリア集中（現在〜6ヶ月）

**選定理由：**
- 成田空港直結・既存インフラ活用
- ドライバーコスト: 月中央値**17万円**（名古屋より11%低い）
- Amazon千葉3拠点（成田・船橋・市原）= Flexドライバー採用源
- 固定便モデルとして最適（ルートが最もシンプル）

**千葉エリアの特性：**

| 項目 | 数値 |
|---|---|
| 月間走行距離 | 3,500 km |
| 燃費 | 10 km/L（渋滞型） |
| ガソリン代/月 | 30,000〜55,000円 |
| 月間経費中央値 | **170,000円** |
| 駐車リスク | 中〜高（住宅密集地） |

### Phase 2: 羽田・東京都心（6〜18ヶ月）

- 成田モデルを羽田起点で複製
- 都内ホテル密度で集約効率UP
- 月150〜200便体制へ

### Phase 3: 名古屋（中部国際空港）展開（18〜36ヶ月）

**名古屋エリアの特性：**

| 項目 | 数値 |
|---|---|
| 月間走行距離 | 4,500 km（郊外型） |
| 燃費 | 13 km/L |
| ガソリン代/月 | 45,000〜70,000円 |
| 月間経費中央値 | **190,000円** |
| 高速代（チャーター時）| 15,000〜40,000円 |
| 固定便特性 | トヨタ系・医療機器など法人需要が厚い |

## 6. 競合比較

### 直接競合

| 競合 | 強み | 弱み | KAIROXの優位 |
|---|---|---|---|
| **ヤマト宅急便（空港宅配）** | 知名度・信頼性 | 翌日配送・カウンター締め切り・多言語非対応 | 当日配送・スマホ完結 |
| **JAL ABC（手荷物宅配）** | 航空会社連携 | 翌日配送・日本語のみ | 当日・4言語 |
| **コインロッカー** | 安価・即時 | ホテル配送なし・預けるだけ | ホテル直接配達 |
| **Klook手荷物保管** | OTA経由で予約しやすい | 保管のみ・配送なし | 配送サービス |

### 間接競合（代替手段）

- 電車で自力搬送（体力的負担・観光時間ロス）
- 空港タクシー（高コスト・スーツケース数に制限）
- ホテルのコンシェルジュ手配（不透明・割高）
