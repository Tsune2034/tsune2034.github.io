# KAIROX 用語集

アプリ・資料・エージェントで使われている英語・カタカナ用語の日本語解説。
新しい用語が増えたら追記する。

---

## サービス・予約

| 用語 | 読み | 意味 |
|---|---|---|
| Booking | ブッキング | 予約。荷物の集荷・配達依頼のこと |
| Matching | マッチング | 依頼者とドライバーを自動で対応させる仕組み |
| Hand Carry / ハンドキャリー | — | 個人が自分の手荷物として他人の荷物を運ぶこと。KAIROXではプラットフォームとして仲介のみ行う |
| Driver / ドライバー | — | 荷物を実際に運ぶ配送担当者 |
| Status / ステータス | — | 予約・配送の現在の進行状態（受付中・配送中・完了など）|
| Live / ライブ | — | リアルタイム追跡が進行中の状態 |
| Review / レビュー | — | 配送完了後にユーザーが入力する評価・口コミ |
| Trust Score | トラストスコア | ドライバーの信頼度を数値化したスコア |
| ETA | イーティーエー | Estimated Time of Arrival の略。到着予定時刻 |
| Cancel / キャンセル | — | 予約の取り消し |
| Terminal / ターミナル | — | 空港の搭乗棟（第1・第2・第3ターミナル）|
| Counter / カウンター | — | 空港の窓口。KAIROXは「カウンター不要」を差別化点とする |
| Check-in / チェックイン | — | ホテルへの入室手続き |
| Player / プレイヤー | — | 追跡画面でのドライバーの表示名称（ゲーム的UI由来）|

## 決済・外部サービス

| 用語 | 読み | 意味 |
|---|---|---|
| Stripe | ストライプ | オンライン決済サービス。クレジットカード・Apple Pay等に対応 |
| JPYC | ジェーピーワイシー | 日本円に連動した暗号通貨（ステーブルコイン）。Phase2で導入予定 |
| Resend | リセンド | メール送信を担うクラウドサービス。確認メールなどに使用 |
| Aviationstack | アビエーションスタック | フライト情報をリアルタイムで取得する外部API |
| Stripe Connect | — | ドライバーへの報酬送金に使うStripeの機能 |

## システム・技術

| 用語 | 読み | 意味 |
|---|---|---|
| GPS | ジーピーエス | 衛星を使った位置情報システム。ドライバーの現在地追跡に使用 |
| API | エーピーアイ | アプリケーション間のデータやり取りの接続口 |
| PIN | ピン | 暗証番号。管理者・ドライバー画面のログイン認証に使用 |
| Admin / 管理者 | — | 管理者専用画面。予約・ドライバー管理・GPS学習などを行う |
| Frontend / フロントエンド | — | ユーザーが見る画面側のプログラム（Next.js） |
| Backend / バックエンド | — | サーバー側のプログラム（FastAPI）。予約・GPS・メール処理などを担当 |
| Deploy / デプロイ | — | プログラムを本番環境に公開する作業 |
| Route / ルート | — | APIのURLパス、または配送の経路 |
| Fallback / フォールバック | — | エラーや障害時に切り替わる代替処理 |
| Alert / アラート | — | 画面上の警告・通知表示 |
| Chat / チャット | — | リアルタイムのテキストメッセージ機能。AIサポートに使用 |
| Railway | レイルウェイ | バックエンドをホスティングするクラウドサービス |
| Vercel | バーセル | フロントエンドをホスティング・デプロイするクラウドサービス |

| 用語 | 読み | 意味 |
|---|---|---|

| Content / コンテンツ | — | SNS投稿・動画・記事などの情報素材 |
| Phase / フェーズ | — | 開発・展開の段階。Phase1＝現在の成田テスト期 |
