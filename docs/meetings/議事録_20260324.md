## 7. task:8 — 貨物運送 法的要件確認（Lawyer エージェント）

### 分析結果

#### Model①：車配送
| 車両 | 要件 |
|------|------|
| 軽自動車 | **黒ナンバー取得**（届出のみ・許可不要） |
| 普通車・トラック | 緑ナンバー（国交大臣許可必要） |
| 白ナンバーで有償 | **違法**（懲役3年リスク） |

#### Model②：ハンドキャリー
- 貨物自動車運送事業法は「自動車使用」が前提 → **徒歩・電車には不適用**
- プレイヤーを「個人事業主（業務委託）」として設計すれば労働者派遣法も回避可能
