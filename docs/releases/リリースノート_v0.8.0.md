# KAIROX リリースノート v0.8.0

**リリース日:** 2026-03-21

---

## v0.8.0 — 管理画面・成田プラン・ドライバー承認・AI予約確認・LINE通知

---

### 新機能: AI予約確認メッセージ + LINE通知（バックエンド）

- 予約作成時（`POST /bookings`）に Claude haiku-4-5 が日本語・英語で確認メッセージを自動生成
- LINE Notify 経由でドライバーおよびオペレーターに即時通知
- `CLAUDE_API_KEY` / `LINE_NOTIFY_TOKEN` 未設定時はスキップ（サービス継続性を保証）
- `backend/main.py` に `generate_confirmation_message()` 関数追加

---

### 新機能: 管理者画面（AdminView）

- `frontend/app/AdminView.tsx` 追加
- 予約一覧・ステータス管理・ドライバー割り当て UI
- `frontend/app/api/admin/route.ts` バックエンドプロキシ追加

---

### 新機能: 成田プラン（/narita）

- `frontend/app/narita/page.tsx` — 成田特化ランディングページ
- `frontend/app/narita/status/[id]/page.tsx` — 成田便専用追跡画面
- `frontend/app/api/flights/route.ts` — フライト情報取得 API プロキシ

---

### 新機能: ドライバー登録・承認フロー

- `frontend/app/driver/approve/[id]/page.tsx` — ドライバー承認ページ（管理者向け）
- `frontend/app/api/driver-register/route.ts` — ドライバー登録エンドポイント
- `frontend/app/api/drivers-active/route.ts` — アクティブドライバー一覧エンドポイント
- GPS近接テロップが実ドライバーデータと連動可能に（Phase 1 完了）

---

### 新機能: LPページ（/lp）

- `frontend/app/lp/page.tsx` — 旅行者向けランディングページ
- SEO・コンバージョン最適化レイアウト

---

### その他

- `frontend/app/i18n.ts` — 4言語（EN/JA/ZH/KO）対応キーを大幅拡充
- `frontend/app/Dashboard.tsx` / `BusinessView.tsx` / `DriverView.tsx` — UI改善・多言語対応強化
- `backend/database.py` — スキーマ更新（ドライバー登録・フライト情報対応）
- `tests/test_briefing.py` — テスト拡充
- 不要な public/ 静的ファイル削除（Next.js デフォルトアセット類）
