### 4. 乗り合わせ（相乗り）機能
`commit 043427a`

**該当コンポーネント:** `RideShareTicker`、`RideshareToast`、目的地カードバッジ

3つのUI要素で乗り合わせをリアルタイムに見せるシミュレーション機能:

| UI | 説明 |
|----|------|
| テロップ（RideShareTicker） | ヘッダー直下に横スクロール。「🚐 3人相乗り · 新宿 · 節約 ¥1,800」を流す |
| トースト（RideshareToast） | 起動5秒後から12秒ごとに左下にポップアップ。行き先ボタンで即選択可能 |
| カードバッジ | 目的地選択の各カードに「🚐 3人相乗り」「節約 ¥1,800」バッジを表示 |
