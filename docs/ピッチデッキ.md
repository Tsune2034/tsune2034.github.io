## スライド 3｜解決策：KAIROXプラットフォーム

### 外国人旅行者には

> 「飛行機が着いた瞬間から始まる。荷物はホテルへ。4言語AIサポート。両替不要」

### 軽貨物ドライバーには

> 「外国語不要。フライト着陸と同時に案件が来る。売上の85〜90%が手元に残る」

### 3つの課題を1つのシステムで同時解決

```
外国人旅行者  →   KAIROXアプリ   →   近くの軽貨物ドライバー
（荷物がある）  （フライト連動・GPS）  （空走中 → 着陸と同時に案件）
                       ↑
              AIがルート最適化・相乗りマッチング
```

### KAIROXの2本柱：車配送 ＋ ハンドキャリー委託

```
【柱①】軽貨物車配送（直接サービス）
  空港 / ホテル → ホテル / 空港 当日配送
  KAIROX社員+個人ドライバーが担当

【柱②】ハンドキャリー委託マッチング（プラットフォーム）
  旅行者 ← GPS近接マッチング → 副業プレイヤー（ギグワーカー）
  KAIROXは仲介のみ（在庫ゼロ・雇用ゼロ）
  手数料：25%（プレイヤー取り分75〜78%）
```

| 荷物量 | スーツケース複数・大型 | 手荷物・小型 |

## スライド 4｜プロダクト：アプリ使用説明

### A. 旅行者向け予約フロー（5ステップ）

**Step 1｜連絡先入力**
- 名前・メール・電話番号を入力
- 右上の言語ボタンで EN / 日本語 / 繁中 / 한국어 を切替
- 予約確認メールが自動送信（4言語・KRX番号付き）

**Step 2｜荷物情報**
- プランを選択：Solo（スーツケース1個）/ Pair（2個）/ Family（4個）
- 追加荷物（土産袋）は1個¥1,500で追加可能
- 相乗りON → AIが同方向の旅行者とマッチングして最大15%割引
- フライト番号を入力 → 急行スロット自動提案（+¥1,000で優先対応）

**Step 3｜配送詳細**

集荷スポットを選択：

*成田エリア（2026-04-01 追加）*
- 成田国際空港 T1・T2 到着ロビー

*北海道エリア（4ゾーン）*
- 千歳：新千歳国際線到着ロビー / 手荷物受取 / 国内線無印良品前 / レラ
- 札幌：ステラプレイス / 大通公園観光案内所 / 狸小路
- 小樽：小樽運河観光案内所 / 堺町通りメルヘン交差点
- 富良野：旭川空港到着ロビー / 旭川駅 / 富良野駅

配送先：ホテル（室内指定可） / 空港（出発便連動）
ホテル受け取りスポット：フロントカウンター / ベルデスク / 正面玄関 / 客室前

**Step 4｜お支払い**

料金内訳がリアルタイム表示：
- 基本料金（Solo ¥3,500〜 / Pair ¥6,000〜 / Family ¥10,000〜）
- 急行保証料（+¥1,000）
- 追加荷物料（+¥1,500/個）
- 相乗り割引（−15%）
- 空港GPS割引（−¥500：空港15km圏内を自動検出）
- 近接ドライバー割引（−¥300：サービスゾーン30km圏内）

支払い方法：
- クレジットカード
- **JPYC**（1 JPYC = ¥1 固定・Polygon・0%手数料・両替不要）
- **USDC**（1 USDC ≈ ¥153・Solana）

**Step 5｜確定・追跡番号発行**
- `KRX-XXXXXX` 形式の追跡番号が即時発行
- AIが相乗りマッチング結果を表示（マッチ人数・最適ルート理由・ETA）
- 予約確認メール自動送信（Resend・4言語）
- キャンセルポリシー：10分以内全額返金 / 10分後90%返金 / 集荷後返金不可

### B. リアルタイム追跡フロー

「配送を追跡」タブ → 追跡番号 `KRX-XXXXXX` を入力

表示される情報：
- **ステータスタイムライン**：予約確認 → 集荷中 → 配送中 → 配達完了
- **ドライバーGPSカード**（Google Mapsリアルタイム表示）：
  - 距離（km）と ETA（分）をリアルタイム表示
  - 出発地・現在地・目的地（空港）が地図上に表示
  - ドライバーステータス：「向かっています / 近くにいます / 到着」
  - 30秒ごとに自動更新（LIVE表示）
- **双方向プリセットチャット**（万が一の緊急連絡）：
  - 顧客→ドライバー：「もうすぐ出ます」「荷物は赤いスーツケース」など4種
  - NEWバッジで未読通知（アンバー点滅）

### C. ドライバーフロー（業界初：フライト連動型）

ヘッダー右端の **🚐** ボタン → PIN入力でログイン

1. 追跡番号（`KRX-XXXXXX`）を入力して追加
2. **✈️ フライトボード** でリアルタイム到着便を監視：
   - フライト番号を「監視」登録
   - 着陸20分前 → アンバーアラート「まもなく着陸」
   - **着陸確認 → 緑アラート「🛬 着陸しました！出発してください」**
   - 着陸後45分カウントダウン開始（入国審査目安）
3. GPS追跡トグルをタップ → 30秒ごとに位置を自動送信
4. **ドライブモード**をタップ：
   - 地図が560pxに拡大（ミラーリング・大画面対応）
   - 進行方向UP（ヘディングアップ）自動ON
   - 「Googleマップでナビ開始（成田空港）」ボタン
5. ステータスボタンで進捗を更新：
   - 🚐 向かっています → 📍 近くにいます（空港500m以内で自動切替）→ 📦 荷物を受け取り中 → ✅ 配達完了
6. **顧客からのメッセージ**をリアルタイム受信（30秒ポーリング）：
   - NEWバッジ（アンバー点滅）で見逃し防止
   - **ドライバー→顧客** 定型返信3種：「今着きました」「渋滞中・10分」「お客様が見当たらない」
7. **証跡写真撮影**：受取時・配達完了時の画像を自動アップロード

### E. GPS学習システム（時間とともに精度が向上）

配送データが蓄積されるほど、KAIROXのETA予測が改善するモート構造：

```
配送完了 → GpsTrackPoint 蓄積（30秒/点）
    ↓
_analyze_route_on_complete() → RouteStats テーブル更新
    ↓
セグメント別補正係数（correction_factor）が計算される
    ↓
次回の同ルートで精度向上したETA提示
```

- 2026-04-01 成田テスト走行: 初記録（16サンプル・23セグメント）
- 走行履歴が増えるほど競合が真似できない精度優位が生まれる

> **注：** 空港手荷物カウンターは「保管のみ」でホテル配送なし。フライト連動・GPS追跡・プリセットチャット・暗号資産決済を全て備えるのはKAIROXのみ。

### 競合が真似できない理由

1. **フライト連動配送**：Aviationstack APIと連携した着陸検知→即時アラートは設計思想から異なる
2. **GPS学習モート**：走行データが蓄積するほど精度が上がる。後発者は追いつけない
3. **ハンドキャリー×車配送 二刀流**：大荷物は車、小荷物はハンドキャリーでプラットフォームが完結。他社は片方のみ
4. **JPYC決済**：前払式支払手段（資金決済法）として法的整理済み。大手は法務コストを嫌う
5. **AI Dispatcher**：GPS近接マッチング→信頼スコア加重→最適プレイヤー自動アサイン（3km以内）
6. **双方向プリセットチャット**：走行中でも安全に使える定型文コミュニケーション
7. **信頼スコア・ランクシステム**：実績が積まれるほどプレイヤー単価が上がる→高品質者が集まる好循環
