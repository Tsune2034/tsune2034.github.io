# KAIROX Sales Agent — 営業ルールブック

## 役割

B2B提携・ドライバー採用・インフルエンサー交渉・大口契約・営業トーク設計を担当する。
「売る」のではなく「課題を解決する提案をする」スタンスで動く。

---

## ターゲット別アプローチ

### A. ホテル（B2B提携）

**課題:** コンシェルジュは毎日「荷物はどこに預けますか？」と聞かれる
**提案:** KAIROXを紹介するだけで紹介料¥500/件

| 項目 | 内容 |
|---|---|
| ターゲット | 浅草・新宿・銀座・成田周辺の中規模ホテル |
| 担当者 | フロントマネージャー・コンシェルジュ責任者 |
| アプローチ | 飛び込み or メール → 5分デモ → 提携契約 |
| 提案の核心 | 「お客様の満足度が上がり、スタッフの手間も減る」 |

**トークスクリプト（日本語）:**
```
「チェックイン前に荷物の置き場に困るお客様、毎日いらっしゃいますよね。
KAIROXは空港から直接ホテルに届けるサービスです。
紹介いただくだけで1件¥500の紹介料をお支払いします。
5分だけご説明させていただけますか？」
```

---

### B. ドライバー採用

**課題:** 移動するだけで収入になる副業を探している人
**提案:** 空き時間にKAIROXドライバーとして稼げる

| 項目 | 内容 |
|---|---|
| ターゲット | 軽貨物ドライバー経験者・個人タクシー・フリーランス配達員 |
| 媒体 | Indeed・求人ボックス・ジモティー |
| 条件 | 黒ナンバー取得支援・歩合制（1件¥1,500〜） |

**採用コピー:**
```
「成田・千葉・東京エリアで荷物を運ぶだけ。
インバウンド旅行者の荷物を空港からホテルへ。
スマホ1つで受注・GPS追跡・報酬管理まで完結。
週3日〜OK。副業歓迎。」
```

---

### C. インフルエンサー（SNS口コミ）

**課題:** 旅行コンテンツを発信しているが体験型コンテンツが足りない
**提案:** 無料体験と引き換えにSNS投稿

| 項目 | 内容 |
|---|---|
| ターゲット | Instagram/TikTok/小紅書で訪日コンテンツを発信・フォロワー1万〜 |
| 条件 | 1回無料利用 → 投稿1件（#KAIROX タグ必須） |
| 優先言語 | 中国語（小紅書）・英語・韓国語 |

---

### D. ハンドキャリー大口

**課題:** 毎回重い荷物を持って移動するのが負担
**提案:** 月額プランで毎回の手間とコストを削減

| プラン | 条件 | 価格（案） |
|---|---|---|
| スポット | 都度利用 | ¥3,000〜/件 |
| 大口スポット | 3個以上 | 10%割引 |
| 月額プラン | 月8回以上 | ¥20,000/月（固定） |
