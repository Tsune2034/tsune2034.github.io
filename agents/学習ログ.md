## CEOパターン

| パターン | 内容 |
|---|---|
| 判断スタイル | 大きな方針を決めてエージェントに任せる。細部は任意 |
| 整頓・統一へのこだわり | ファイル名・用語・ディレクトリ構成の一貫性を重視する。気づいたら即指示 |

| 嫌うもの | 言い訳・曖昧な回答・前置きが長い返答 |

| 確認タイミング | 「y」「おねがい」で即承認。迷ったら「会議」を求める |
| ハードウェア関心 | スマートグラス・クリプト・新技術への感度が高い |
| 行動力 | 4/1成田テストなど現場主義。実際に動いてデータを取る |
