## 1. エグゼクティブサマリー

KAIROXは、北海道・千歳エリアを起点とした**インバウンド観光客および半導体・IT企業出張者向け急行荷物配送サービス**である。

黒ナンバー軽バン4WD × 道トクふりーぱす × AIルート最適化の組み合わせにより、ヤマト運輸等の大手が構造的に真似できない「分単位の時間保証・多言語AI対応・相乗りマッチング」を提供する。ラピダスを核とした千歳の産業集積が生み出す法人需要と、増加するインバウンド観光需要の両輪で成長を図る。

---

## 2. ポジション定義

```
「千歳・新千歳を中心とした
 インバウンド × ビジネス特化の
 急行配送プラットフォーム」

競合が取れないポジション:
  多言語AI × アプリ × 急行便 × 法人対応
  × 黒ナンバー合法 × 4WD北海道全天候対応

競合分析:
  ヤマト運輸 → 翌日以降・観光特化なし・多言語ゼロ
  地場業者   → アプリなし・多言語なし
  タクシー   → 荷物のみ配送は法的に不可
```

## 10. 北海道エアポート（HAP）連携戦略

### 概要

新千歳空港を運営する**北海道エアポート株式会社（HAP）**との連携は、KAIROXの集客・認知度向上において最重要の営業アクションである。空港内に物理的な接点を持つことで「到着した観光客が即座にKAIROXを知る」動線を作れる。

### 出店形態と目的

| 形態 | 場所 | 目的 | 優先度 |
|------|------|------|--------|
| 催事・イベントブース | 国際線・国内線ターミナル内 | 期間限定PR・サービス説明 | ★★★ 即時 |
| 常設カウンター | レンタカー・旅行代理店エリア | 長期的な予約受付・荷物引受 | ★★☆ Phase 1 |
| デジタルサイネージ | 到着ロビー・手荷物受取エリア | QRコード誘導・多言語案内 | ★★☆ Phase 1 |

### 申請手順

```
STEP 1: 事前相談（今すぐ）
  → 北海道エアポート株式会社 コーポレートサイトから
    出店・利用に関する問い合わせ窓口へ接触
  → 「インバウンド向け手ぶら配送サービスの
     PR拠点として空港内スペースを活用したい」旨を伝える

STEP 2: イベントスペースから開始
  → 株式会社えんれいしゃ経由でターミナル内
    イベントスペースの仮予約・詳細確認
  → 期間限定ブース（週末・繁忙期）でまず実績を作る

STEP 3: 空港供用規程に基づく正式申請
  → 第2条・第4条等に基づき必要書類提出
  → 運営側の審査を経て許可取得

STEP 4: 常設カウンター申請（Phase 1）
  → ブース運営の実績・利用者実績をもとに
    賃貸借契約へ移行
```

### 設置場所の候補優先順位

```
優先度 ★★★:
  国際線ターミナル 到着ロビー
  → 外国人観光客が最初に目にする場所
  → 多言語案内 + QRコードでその場で予約

優先度 ★★☆:
  手荷物受取エリア付近
  → 「この荷物、ホテルまで届けます」の訴求に最適
  → 視覚的インパクト大

優先度 ★☆☆:
  国内線ターミナル
  → 国内出張者・ビジネス客向け
  → ラピダス関連出張者への認知拡大
```

### 営業所ブース獲得資金の目安

| フェーズ | 形態 | 費用目安 | 備考 |
|----------|------|----------|------|
| Phase 0（即時） | 催事ブース（週末1〜3日） | ¥5,000〜¥30,000/回 | えんれいしゃ経由。繁忙期（GW・夏・年末）は割増 |
| Phase 0（週次） | 定期イベントスペース | ¥5万〜¥15万/月 | 面積・場所・期間により変動。1〜3㎡程度想定 |
| Phase 1（常設） | 常設カウンター（賃貸借） | ¥10万〜¥50万/月 | 国際線ロビー近辺は高め。実績積み上げ後に交渉 |
| Phase 1（看板） | デジタルサイネージ枠 | ¥5万〜¥20万/月 | HAP広告枠または外部OOH業者経由 |

**初期費用の試算（Phase 0 催事出展スタート）:**
```
催事ブース什器（テーブル・サイネージ・のぼり等）  ¥30,000〜¥80,000
初回ブース出展費（2日間）                          ¥10,000〜¥30,000
多言語チラシ印刷（英・日・中・韓 各200部）         ¥15,000〜¥30,000
──────────────────────────────────────────────────
合計（概算）                                        ¥55,000〜¥140,000
```

**資金調達の優先順位:**
1. 催事ブース出展実績を3〜6回積み上げる（月1回ペース・黒字化前提）
2. 月次法人契約（ラピダス系1〜2社）で固定収入を確保してから常設に移行
3. 常設カウンターは「ブース運営実績 × 月間利用者数データ」でHAPと交渉

### 注意事項

```
・運営状況は流動的。修繕工事等で利用制限が発生する場合あり
・最新の空港供用規程は HAPコーポレートサイトで都度確認
・イベントスペース詳細（場所・面積）は えんれいしゃ案内資料を参照
・空港管理に支障がある場合は利用制限・休止の可能性あり
```

### HAPとの連携がもたらす効果

```
Before（アプリのみ）:
  観光客がKAIROXを知るには SNS・検索が必要

After（HAP連携）:
  到着した瞬間にKAIROXのカウンターが目の前にある
  → コンバージョン率 推定3〜5倍
  → 「空港公認」のブランド信頼感
  → 競合が後追いできないロケーション優位
```

## 11. JPYC × USDC 決済戦略 — メディア・PR角度

### なぜJPYCか

| 観点 | 内容 |
|------|------|
| 法規制 | 前払式支払手段（資金決済法）として合法。暗号資産非該当 |
| 為替リスク | 1 JPYC = ¥1 固定。カスタマーにとってUSDCより分かりやすい |
| 国内B2B適合 | ラピダス・半導体装置メーカーの経理処理がしやすい |
| メディア訴求 | 「日本のステーブルコインを使った観光配送」はWeb3系・観光系両方で報道価値大 |

| USDC | Solana | 0% | インバウンド外国人 |
