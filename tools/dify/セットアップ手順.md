# Dify × KAIROX セットアップ手順

## 概要

DifyはAIエージェントをGUIで設計・公開できるプラットフォーム。
KAIROXの13エージェントをDifyで管理することで、将来のクライアントへの提示が容易になる。

```
現在: CLAUDE.md に13エージェントを定義（エンジニア向け）
目標: Dify GUIでエージェントを設計 → クライアントへのデモに使える
```

---

## Step 1: Difyアカウント作成（クラウド版・無料）

1. https://dify.ai にアクセス
2. 「Get Started for Free」→ Googleアカウントでログイン
3. ワークスペース名: `KAIROX Intelligence`

> ローカルホスト版より**クラウド版を推奨**（Railway不要・すぐ使える）

---

## Step 2: Claude APIキーを設定

1. Dify管理画面 → Settings → Model Provider
2. 「Anthropic」を選択
3. APIキーを入力
4. デフォルトモデル: `claude-sonnet-4-6`（最新）

---

## Step 3: KAIROXエージェントを作成

以下の順で作成する（優先度順）:

### 3-1. サポートエージェント（最初に公開するもの）

アプリ種別: **Chatbot**

```
System Prompt:
あなたはKAIROX（日本の手荷物配送サービス）のカスタマーサポートAIです。
以下の言語で対応してください: 英語、日本語、中国語（簡体字）、韓国語。
---
サービス概要: 成田空港で荷物を預かり、宿泊ホテルに配送します。
料金: スタンダード ¥3,000 / プレミアム ¥5,000
対応時間: 7:00-22:00
---
丁寧で簡潔に回答し、不明な点はsupport@kairox.jpへ案内してください。
```

### 3-2. データアナリストエージェント

アプリ種別: **Agent**（ツール付き）

```
System Prompt:
あなたはKAIROXのデータアナリストです。
予約数・売上・コンバージョン率を分析し、意思決定を支援します。
数値サマリー・原因仮説・アクション提案の順で回答してください。
```

ツール設定:
- HTTP Request（KAIROXのAPIから統計取得）

### 3-3. ストラテジーエージェント

アプリ種別: **Agent**

```
System Prompt:
あなたはKAIROXの戦略アドバイザーです。
成長戦略・事業拡大・意思決定を支援します。
インバウンド市場のトレンドを踏まえ、具体的な行動計画を提示してください。
```

---

## Step 4: KAIROX Council パイプラインを構築

アプリ種別: **Workflow**

```
ユーザー入力
    ↓
意図分類（LLMノード）
    ↓
Switch（support / data / strategy / engineer）
    ↓
各エージェント（それぞれLLMノード）
    ↓
出力結合
    ↓
回答
```

この「KAIROX Intelligence Council」ワークフローを完成させると、
クライアントへのデモで「AIがチームとして動く」姿を見せられる。

---

## Step 5: APIとして外部公開（オプション）

Dify → アプリ → API Access
→ エンドポイントURLとAPIキーをコピー

KAIROXアプリの `/api/company` にDify APIを組み込む:

```typescript
// app/api/company/route.ts（将来対応）
const response = await fetch('https://api.dify.ai/v1/chat-messages', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${process.env.DIFY_API_KEY}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    inputs: {},
    query: userInput,
    response_mode: 'blocking',
    user: 'kairox-council',
  }),
});
```
