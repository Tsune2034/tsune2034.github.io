## スライド 2｜解決する2つの問題

### 問題① 外国人旅行者の荷物問題

```
新千歳空港に着いた台湾人4人家族。
スーツケース4個＋土産袋8個を抱えて
バスに乗り、ホテルに運ぶ。
→ 観光時間の30%が荷物運搬で消える。
```

- 北海道訪問外国人：**年間300万人超**（2024年回復基調）
- 富良野外国人宿泊客：前年比 **+50%** の急成長
- 既存のヤマト翌配では「今日中に届けてほしい」に応えられない

### 問題② 軽貨物ドライバーの搾取構造

```
Amazon Flex・大手下請けの実態：
売上の30〜50%が中間マージンとして抜かれる。
配送完了後の帰路は毎回空車（空走）。
外国人客に対応できず、インバウンド需要を取れない。
```

- 軽貨物ドライバーの平均年収：約350〜450万円
- 大手還元率：60〜70%
- 空走率：推定30〜40%（帰路の燃料費・時間が丸損）

## スライド 4｜プロダクト：アプリ使用説明

### A. 旅行者向け予約フロー（5ステップ）

**Step 1｜連絡先入力**
- 名前・メール・電話番号を入力
- 右上の言語ボタンで EN / 日本語 / 繁中 / 한국어 を切替

**Step 2｜荷物情報**
- プランを選択：Solo（スーツケース1個）/ Pair（2個）/ Family（4個）
- 追加荷物（土産袋）は1個¥1,500で追加可能
- 相乗りON → AIが同方向の旅行者とマッチングして最大15%割引
- フライト番号を入力すると急行スロットを自動提案（+¥1,000で優先対応）

**Step 3｜配送詳細**
- 集荷スポットをタップ選択：
  - 千歳エリア：新千歳国際線到着ロビー / 手荷物受取 / 国内線無印良品前 / レラ
  - 札幌エリア：ステラプレイス / 大通公園観光案内所 / 狸小路
  - 小樽エリア：小樽運河観光案内所 / 堺町通りメルヘン交差点
  - 富良野エリア：旭川空港到着ロビー / 旭川駅 / 富良野駅
- 配送先を選択：ホテル / 新千歳空港（成田・羽田は近日対応）
- ホテル選択 → ゾーン別ホテルリストから検索（全4ゾーン・40軒以上）
- ホテル内受け取りスポット：フロントカウンター / ベルデスク / 正面玄関 / 客室前

**Step 4｜お支払い**
- 料金内訳がリアルタイム表示：
  - 基本料金（Solo ¥3,500〜 / Pair ¥6,000〜 / Family ¥10,000〜）
  - 急行保証料（+¥1,000）
  - 追加荷物料（+¥1,500/個）
  - 相乗り割引（−15%）
  - 空港GPS割引（−¥500：新千歳15km圏内を自動検出）
  - 近接ドライバー割引（−¥300：サービスゾーン30km圏内）
- 支払い方法を選択：
  - クレジットカード
  - **JPYC**（1 JPYC = ¥1 固定・Polygon・0%手数料・両替不要）
  - **USDC**（1 USDC ≈ ¥153・Solana）

**Step 5｜確定・追跡番号発行**
- `KRX-XXXXXX` 形式の追跡番号が即時発行
- AIが相乗りマッチング結果を表示（マッチ人数・最適ルート理由・ETA）
- キャンセルポリシー：10分以内全額返金 / 10分後90%返金 / 集荷後返金不可

---

### B. リアルタイム追跡フロー

「配送を追跡」タブ → 追跡番号 `KRX-XXXXXX` を入力

表示される情報：
- **ステータスタイムライン**：予約確認 → 集荷中 → 配送中 → 配達完了
- **ドライバーGPSカード**（ドライバーがGPSをONにしている場合）：
  - 距離（km）と ETA（分）をリアルタイム表示
  - 集荷地点→配送先のプログレスバー
  - ドライバーステータス：「向かっています / 近くにいます / 到着」
  - 10秒ごとに自動更新（LIVE表示）

---

### C. ドライバーフロー

ヘッダー右端の **🚐** ボタン → PIN入力でログイン

1. 追跡番号（`KRX-XXXXXX`）を入力して「追加」
2. GPS追跡トグルをタップ → ブラウザのGPS権限を許可
   - 30秒ごとに位置を自動送信
   - タップで停止・再開可能
3. ステータスボタンで進捗を更新：
   - 🚐 向かっています → 📍 近くにいます → 📦 荷物を受け取り中 → ✅ 配達完了
   - ステータス更新時に位置情報も即時送信
4. 複数の配送を同時管理可能（案件ごとにカード表示）

4. **GPSリアルタイム**：ドライバースマホGPS→顧客へのLIVE追跡。既存サービスは翌日確認のみ
5. **GPS近接割引**：リアルタイム需要創出メカニズム。手荷物預かり所には存在しない概念

## スライド 7｜ビジネスモデル

### 収益構造：ハイブリッドスケールモデル

社員ドライバーによる品質基盤を固めながら、個人ドライバーと法人契約で拡大速度を最大化する。

**Phase 1（現在）：CEO自走型**
```
売上 = 配送料全額（¥3,500〜¥15,000/件）
コスト = 燃料費・消耗品・アプリ運営費
利益率 = 60〜73%
目的 = 採算検証・現場ノウハウ蓄積
```

**Phase 2（3〜6ヶ月）：社員採用 × 個人登録 並走**
```
社員ドライバー（2〜3名）
  → 品質管理・法人対応・ブランド担保
  → 固定費：給与＋社会保険（月20〜30万/人）
  → 法人サブスク収益（¥50,000〜/月×3社）で社員コストをカバー

個人ドライバー（登録制・5〜10名）
  → ピーク対応・エリア拡張・固定費ゼロ
  → 還元率85〜90%・GPS近接マッチングで稼働最大化
```

**Phase 3（12ヶ月〜）：地域フランチャイズ型**
```
北海道各エリア（富良野・旭川・帯広・函館）に
社員1名＋登録ドライバー複数の「ハブ」を設置
法人・ホテル・空港との定期契約で安定収益を確保
```

> 法人契約収益で社員給与を賄い、個人登録ドライバーで変動コストを最小化。
> 「社員が品質の柱、個人が成長の翼」という二層構造。
