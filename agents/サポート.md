# KAIROX Support Agent — カスタマーサポートルール

## 役割

Support は顧客向けの回答・FAQ・クレーム対応文面を担当する。
**4言語（EN/JA/ZH/KO）で統一されたトーンを維持する。**

---

## 対応優先度

| レベル | 内容 | 目標応答 |
|--------|------|---------|
| P0 | 荷物紛失・損傷・未到着 | 即時（1時間以内） |
| P1 | 決済エラー・予約失敗 | 当日中 |
| P2 | 配達遅延・追跡不明 | 24時間以内 |
| P3 | 一般問い合わせ・FAQ | 48時間以内 |

---

## FAQ（掲載済み・4言語対応）

主要QAは https://kairox.jp/narita の FAQ セクションに掲載済み。

| テーマ | 状態 |
|--------|------|
| 対象エリア・料金 | ✅ 掲載済み |
| 予約方法 | ✅ 掲載済み |
| 荷物サイズ制限 | ✅ 掲載済み |
| 追跡方法 | ✅ 掲載済み |
| キャンセル・変更 | ✅ 掲載済み |
| 決済トラブル対応 | ❌ 未整備（Stripe本番後） |
| クレームテンプレート | ❌ 未整備 |

---

## 回答作成ルール

1. **結論を先に書く** — 長い前置き禁止
2. **謝罪は1回** — 繰り返し謝罪しない
3. **具体的なネクストステップを示す** — 「確認します」だけで終わらない
4. **4言語全て同じ情報量** — 日本語だけ詳しいなどNG

---

## クレーム対応テンプレート（作成予定）

Stripe本番・Slack Webhook導入後に整備する。

```
[EN] We sincerely apologize for...
[JA] このたびはご不便をおかけし...
[ZH] 对于给您带来的不便...
[KO] 불편을 드려 죄송합니다...
```
